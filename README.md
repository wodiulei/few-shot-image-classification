# few-shot-image-classification
作为本科毕设论文相关代码的存储仓库，主要存储所用的数据集切割、训练模型、后续做正常实验以及消融实验所用代码及预训练模型相关文件

## 项目代码简单讲解
split_dataset.py：该文件是数据集分割代码，核心功能是将某个数据集按照代码内部的参数设定划分为N way-K shot的数据集。
DINO.py：该文件作为模型训练的主要代码，核心逻辑都在里面（内含三个布尔变量，用于控制对应模块是否生效，做消融实验时设定使用）。
run_ablation.py：该文件是调用训练好后的模型在指定数据集上进行图像分类任务并输出相应准确率。

注：当前项目无法直接运行，由于某些原因，预训练模型需要加载在本地，无法从网络中加载，若需要运行项目，需要自行下载预训练模型，链接地址：https://pan.quark.cn/s/b3456d0cbcc1，下载后放入DINO文件夹中即可

本项目训练模型所使用数据集来自2024年全国大学生计算机能力挑战赛（大数据赛道），若有需要则自行下载（此处提供的是划分好的数据集），地址：https://pan.quark.cn/s/a9ec86b8129e
